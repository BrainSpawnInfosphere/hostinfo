Install
--------

The preferred way is to use `pypi.org <https://pypi.python.org/pypi>`_ ::

    pip install hostinfo

For development you can also do::

    git clone https://github.com/walchko/hostinfo.git
    cd hostinfo
    pip install -e .

------

	infoserver.py --port 8800
